Đây là phần mềm quản lý nhân sự do t tự phát triển. 
Dưới đây là hướng dẫn nhanh để sử dụng:

1. CÁCH KHỞI ĐỘNG PHẦN MỀM
--------------------------
- Tìm file có tên "Konami.exe" (biểu tượng hình logo).
- Nhấn đúp chuột để chạy.
- Đợi khoảng 3-5 giây để phần mềm khởi động.

(*) LƯU Ý QUAN TRỌNG:
- CẦN CÀI ĐẶT Java-25
- TUYỆT ĐỐI KHÔNG được tách riêng file "Konami.exe" ra khỏi thư mục này.
- KHÔNG XÓA file "konami_data.db". Nếu xóa, phần mềm sẽ không chạy được hoặc mất hết dữ liệu.

2. TÀI KHOẢN ĐĂNG NHẬP (Mặc định)
---------------------------------
Nếu phần mềm yêu cầu đăng nhập, hãy sử dụng tài khoản sau:
--Giao diện admin--
- Tên đăng nhập: admin
- Mật khẩu: 123456
--Giao diện staff-- 
- Tên đăng nhập: NV001 (Hoặc NV002, NV003, NV004, ... NV040)
- Mật khẩu: 123

3. CÁC TÍNH NĂNG CHÍNH
----------------------
- Quản lý nhân viên: Thêm, Sửa, Xóa hồ sơ nhân sự.
- Tính lương: Tự động tính lương thực lĩnh dựa trên hệ số và ngày công.
- Thống kê: Xem báo cáo tổng quan về tình hình nhân sự và quỹ lương.
- Xuất Excel: Xuất danh sách nhân viên ra file Excel để in ấn.
- Báo lỗi: Thử bấm vào đó rồi báo cáo đi, t biết hết đấy.

4. HỖ TRỢ
